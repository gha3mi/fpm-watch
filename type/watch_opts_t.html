watch_opts_t

type, public :: watch_opts_t

Watcher configuration options.

Values are typically assembled from: - hard-coded defaults, - fpm.toml overrides ([extra.fpm-watch]), - command-line flags (--watch-*).

Time values (poll, debounce, rescan) are in seconds. Verbosity level: - -1: quiet (errors only) - 0: normal - 1..2: more detail
